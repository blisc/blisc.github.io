Projects

Design Projects

Processor

A lab project as part of my Computer Organization class. I thoroughly enjoyed it as it was my first major verilog project and required conceptualizing the entire processor as a system. We were given a barebones single cycle processor with an ALU, register file, and memory system. We converted it to a multicycle, pipleined processor.
  • Created and designed the data path and control signals of a multicycle, pipelined RISC processor
  • Correctly handles control and data hazards using bubbles and bypass wires respectively
  • One of the 7 fastest processor designs of the class

The Royal Throne

A design project for a robotics class in second year. We were asked to design and build an autonomous robot that would check the state of an UFO Tool Light. The machine had to turn on the Tool Light, check how many of the LEDs were functioning, and then turn them off. Our machine successively checked the state of 9 Tool Lights in one operation.
  • Created an autonomous machine that checked the state of LEDs on an UFO Tool Light
  • Programmed the PIC microcontroller and interfaced with the circuits
  • Designed a simple text user interface and developed the logic and control sequences
  • Determined the state of LEDs from the analog readings of 6 photoresistors
  • Allowed for the storage of previous test logs to the PIC EEPROM
